## Das Kurzzeitgedächtnis des Computers

**Was ist RAM?**
- **Random Access Memory (RAM)** ist der Arbeitsspeicher eines Computers. Er speichert Daten, auf die der Prozessor ([[CPU]]) schnell zugreifen muss.
- Im Gegensatz zu Festplatten oder SSDs ist RAM **flüchtig**. Das bedeutet, dass alle Daten verloren gehen, sobald der Computer ausgeschaltet wird.
- RAM ist essenziell für die Ausführung von Programmen. Beim Starten eines Programms wird es in den RAM geladen, damit die [[CPU]] darauf zugreifen kann.

**Arten von RAM**
- **DRAM (Dynamic Random Access Memory)**: Die gebräuchlichste Art von RAM in Computern.
- **SDRAM (Synchronous DRAM)**: Eine Weiterentwicklung von DRAM, die mit dem Systemtakt synchronisiert ist, was die Datenverarbeitung beschleunigt.
- **DDR SDRAM (Double Data Rate SDRAM)**: Die aktuelle Generation von RAM, die noch schneller und effizienter ist als SDRAM. Es gibt verschiedene Versionen wie DDR3 und DDR4, DDR5 wobei DDR5 die neueste und schnellste ist.

**Wie RAM funktioniert**
1. **Daten speichern:** Wenn du etwas in einem Dokument tippst oder ein Spiel spielst, werden diese Daten im RAM gespeichert.
2. **[[CPU und RAM#Die Kommunikation zwischen CPU und RAM|Daten abrufen:]]** Die [[CPU]] greift auf die Daten im RAM zu, um sie zu verarbeiten und auszuführen.
3. **Flüchtigkeit:** Sobald der Computer ausgeschaltet wird, werden alle Daten im RAM gelöscht.

**Warum ist RAM wichtig?**
- **Geschwindigkeit:** RAM ermöglicht einen schnellen Zugriff auf Daten, was die Leistung des Computers verbessert.
- **Multitasking:** Mit mehr RAM können mehrere Programme gleichzeitig ausgeführt werden, ohne dass der Computer langsamer wird.

**Worauf du beim Kauf von RAM achten solltest**
- **Kompatibilität:** Stelle sicher, dass der RAM mit deinem [[Motherboard]] kompatibel ist (DDR-Version, Speicherkapazität, Anzahl der Pins).
- **Kapazität:** Je mehr RAM du hast, desto mehr Programme kannst du gleichzeitig ausführen. Für anspruchsvolle Aufgaben wie Videobearbeitung oder Spiele sind 16 GB oder mehr empfehlenswert.
- **Geschwindigkeit:** Schnellere RAM-Module (höhere Taktfrequenz) können die Leistung verbessern, aber der Unterschied ist oft gering.

**Zusammenfassung**
RAM ist ein kritischer Bestandteil jedes Computers. Er ermöglicht schnelles Multitasking und die Ausführung anspruchsvoller Programme. Achte beim Kauf auf Kompatibilität und Kapazität, um die optimale Leistung für dein System zu erzielen.
