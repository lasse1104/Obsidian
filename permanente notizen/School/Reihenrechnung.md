# Formeln
**U1 = R1 x I**
**U2 = R2 x I**
**Ug = U1 + U2**
**Rg = R1 + R2**

Beispiele:
R1 = 25 Omega
R2 = 35 Omega
Ug = 220V

Ergebnisse:
Rg = R1 + R2 = 60 Omega
I = Ug/R = 220V/60Omega
